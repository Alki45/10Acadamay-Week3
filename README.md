# AlphaCare Insurance Solutions - Marketing Analytics Project

## Project Overview

This project aims to analyze historical insurance claim data to optimize marketing strategies and identify low-risk targets for premium reductions at AlphaCare Insurance Solutions (ACIS). The analysis will leverage statistical methods and machine learning techniques to deliver actionable insights.

## Objectives

- Understand the insurance claim data and extract valuable insights.
- Perform hypothesis testing to analyze risk differences across various demographics.
- Develop predictive models for insurance claims and optimal premium values.
- Provide recommendations to enhance insurance product offerings.

## Data Description

The dataset contains historical insurance claims data from February 2014 to August 2015, including various features related to:

- **Insurance Policy**: Policy ID, transaction date, underwritten cover ID, etc.
- **Client Information**: Citizenship, gender, marital status, etc.
- **Car Details**: Vehicle type, make, model, registration year, etc.
- **Plan Details**: Premium amounts, coverage types, etc.
- **Claims Information**: Total claims, total premiums, etc.

## Key Features

- **Exploratory Data Analysis (EDA)**: Comprehensive analysis to understand data distributions, correlations, and trends.
- **Hypothesis Testing**: Analyze risk differences using A/B testing methodology.
- **Machine Learning Models**: Implement linear regression and other models to predict claims and premium values.
- **Data Visualization**: Create insightful visualizations to present key findings.

## Technologies Used
